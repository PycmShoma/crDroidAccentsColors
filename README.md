# crDroidAccentThemeColors
- Magisk module for crDroid

## Description
- Changes the theme in some system applications to the accent color of the selected style on Main Settings
- Gestural Navbar (w)
- Messaging app (w)
- Dialer app (w)
- DeskClock app (w)
- MatLog app (w)
- OPScreenRecorder app (w)
- Contacts app (wip)
- DocumentsUI app (wip)
